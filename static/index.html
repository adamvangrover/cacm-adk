Welcome to the CACM-ADK

The CACM Authoring & Development Kit is an intelligent system designed to help users (credit analysts, domain experts, developers) author standardized Credit Analysis Capability Modules (CACMs).

This ADK provides tools and components to define, validate, simulate, and (eventually) execute these modular credit analysis capabilities. The goal is to promote standardization, reusability, and agility in developing and deploying credit analytics.

This page serves as a simple entry point and proof-of-concept demonstration.
